# 🌎 Guess the City Game

**Where in the World Has BASE1 Been?**

An interactive, touch-screen-friendly quiz game featuring photos from BASE1 customer sites and team events around the globe.

## How to Play

1. Tap **Start** to begin
2. Each round shows a photo — guess which city it was taken in
3. Choose from 4 options. Green = correct, red = wrong
4. After each answer, learn the story behind the visit
5. See your score at the end — aim for a perfect 11/11!

## Cities Featured

Cancún · Grand Rapids · Maui · London · Memphis · Québec City · Shanghai (Minhang) · Shanghai (Pudong) · Suzhou · Seattle · Victoria BC

## Hosting

This game is deployed automatically to **GitHub Pages** on every push to `main`.

🔗 **Live URL**: `https://<your-username>.github.io/guess-the-city-game/`

## Running Locally

Just open `index.html` in any browser. No server or dependencies needed — everything is self-contained in a single file.

## For Touch Displays

Open the live URL or local file in a browser and press **F11** for full-screen mode. The interface is optimized for tap interactions with large buttons and no scrolling.
